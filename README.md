# MySQL Migration

轻量，简单又好用的数据库迁移工具。支持创建、修改、数据集填充等基础功能。

### 使用

![usage](docs/2017-04-29%2016.26.07.gif)

##### Dump
 
```
php migrate migrate dump [-p|--path]
```

##### Run

```
php migrate migrate run [-p|--path]
```
